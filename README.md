# 🎲 Dice Rolling Simulator (Python CLI)

A simple command-line dice rolling simulator built in Python that allows users to roll virtual dice multiple times and track total rolls across a session.

---

## ✨ Features

- Roll a standard 6-sided dice (D6)
- Roll multiple dice in one go
- Track total number of rolls in a session
- Replay system with reset option
- Continuous interactive CLI experience
- Session-based statistics

---

## 🛠️ Tech Stack

- Python 3
- Random module

---

## 🧠 How It Works

1. User chooses to start rolling the dice
2. User enters number of rolls per round
3. System generates random values (1–6)
4. Results are displayed in a formatted list
5. Total rolls are tracked across the session
6. User can continue, restart, or exit

---

## 💡 Example Usage

```
Do you want to roll the dice? (y/n): y
How many times you want to roll: 3
You rolled: (4, 1, 6)
Total rolls so far: 3
```
